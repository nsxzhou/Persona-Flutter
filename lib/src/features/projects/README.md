# Projects Feature

Owns local writing projects, project workbench state, chapter tree entry points, and future Zen Editor child routes.

## Layers

* `domain/` — project entities, chapter entities, value objects, and repository contracts.
* `application/` — project use cases, orchestration services, and Riverpod providers.
* `data/` — Drift tables/DAOs, DTOs, mappers, and repository implementations.
* `presentation/` — pages, widgets, controllers, and UI state.
