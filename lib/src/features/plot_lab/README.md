# Plot Lab Feature

Owns plot sample imports, story skeleton extraction, Story Engine generation, and reusable Plot Profiles.

## Layers

* `domain/` — plot profile entities, skeleton models, task contracts, and validation.
* `application/` — plot analysis use cases, task orchestration, and Riverpod providers.
* `data/` — Drift tables/DAOs, local file references, DTOs, mappers, and repositories.
* `presentation/` — Plot Lab pages, dialogs, progress views, and UI state.
