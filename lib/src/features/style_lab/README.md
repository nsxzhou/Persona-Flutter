# Style Lab Feature

Owns style sample imports, style analysis tasks, Voice Profile generation, and reusable Style Profiles.

## Layers

* `domain/` — style profile entities, analysis task contracts, and domain validation.
* `application/` — style analysis use cases, task orchestration, and Riverpod providers.
* `data/` — Drift tables/DAOs, local file references, DTOs, mappers, and repositories.
* `presentation/` — Style Lab pages, dialogs, progress views, and UI state.
