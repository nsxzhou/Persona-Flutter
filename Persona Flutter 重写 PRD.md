# Persona Flutter 重写 PRD

## 1. 产品概述

Persona 是一个单用户、本地优先、BYOK 的 AI 长篇创作平台。  
本次重写目标是用 `Flutter + Dart + SQLite` 重建为桌面优先的本地应用，打开即用，无需登录，无需外部服务常驻。

### 核心关键词
- 单用户
- 本地离线
- BYOK（用户自带 API Key）
- OpenAI-compatible
- SQLite
- 桌面优先：macOS → Windows
- 移动端后置，但最终功能对等

## 2. 产品目标

1. 保留现有全部功能。
2. 统一成一个安装包，启动即进入可用状态。
3. 去掉云依赖、账号体系和跨设备同步。
4. 提升桌面端交互、性能和稳定性。
5. 为后续移动端对等迁移预留架构。

## 3. 范围定义

### 3.1 In Scope
- 本地项目管理
- Provider 配置与测试
- 风格分析（Style Lab）
- 剧情分析（Plot Lab）
- 项目工作台
- Zen Editor 写作
- 章节管理
- 局部改写
- 记忆同步
- 导入 / 导出
- 任务状态管理
- 本地历史记录
- 备份 / 恢复

### 3.2 Out of Scope
- 用户登录
- 多用户协作
- 云同步
- 跨设备实时同步
- 非 OpenAI-compatible Provider
- 远程后端
- Redis / MQ / SaaS 化部署

## 4. 目标用户

- 长篇小说作者
- 独立写作者
- 需要本地控制 API Key 的用户
- 需要大文本分析与辅助写作的重度用户

## 5. 核心用户流程

1. 首次打开 App
2. 自动初始化本地 SQLite 和本地后端
3. 配置 OpenAI-compatible Provider
4. 导入样本文本或新建项目
5. 执行 Style Lab / Plot Lab 分析
6. 挂载分析结果到项目
7. 进入工作台整理设定与大纲
8. 进入编辑器写作、改写、同步记忆
9. 导出文本或备份整个本地数据

## 6. 功能需求

### 6.1 初始化与启动
- App 启动时自动拉起本地后端
- 首次启动自动创建本地数据库
- 无登录、无注册、无账号体系
- 支持本地环境重置与重新初始化

### 6.2 Provider 管理
- 创建、编辑、删除 Provider
- 支持 OpenAI-compatible Base URL
- 支持 API Key 保存
- 支持默认模型配置
- 支持连通性测试
- 支持测试结果展示
- 支持启用 / 禁用 Provider

### 6.3 Style Lab
- 导入 TXT 样本
- 生成风格分析报告
- 生成 Voice Profile
- 保存为可复用的 Style Profile
- 支持任务进度、失败、重试、暂停、恢复

### 6.4 Plot Lab
- 导入 TXT 样本
- 生成全书骨架
- 生成 Story Engine
- 保存为可复用的 Plot Profile
- 支持任务状态管理

### 6.5 Project Workbench
- 创建项目
- 管理项目简介、设定、角色、总纲、分卷、章节细纲
- 挂载 Style Profile / Plot Profile
- 管理章节结构
- 编辑项目蓝图

### 6.6 Zen Editor
- 章节导航
- 正文编辑
- 节拍驱动写作
- 局部改写
- 记忆同步
- 章节状态管理
- 草稿保存与恢复

### 6.7 导入 / 导出 / 备份
- 导入样本文本
- 导入项目备份
- 导出项目内容
- 导出全文文本
- 导出 EPUB
- 导出本地备份包
- 从备份恢复整个本地工作区

### 6.8 任务系统
- 长任务异步执行
- 任务进度可见
- 失败原因可查看
- 支持暂停 / 恢复
- 支持崩溃后恢复

## 7. 数据与架构约束

### 7.1 技术栈
- 前端：Flutter
- 业务层：Dart
- 数据库：SQLite
- 运行形态：单 App + 自动拉起本地后端
- Provider：仅 OpenAI-compatible BYOK

### 7.2 数据策略
- 全部数据本地存储
- 不做跨设备同步
- 只做导入 / 导出备份
- 不做云端账号体系
- 不做多端状态合并

### 7.3 平台策略
- 第一阶段：macOS
- 第二阶段：Windows
- 移动端：后置，但设计上保持功能对等目标

## 8. 非功能需求

- 启动后立即可用
- 支持长文本编辑不卡顿
- 支持大项目数据量
- 支持任务中断恢复
- 支持离线运行
- 支持本地文件导入导出稳定性
- UI 适配桌面窗口缩放
- 后续能平移到移动端

## 9. 术语说明

- BYOK：Bring Your Own Key，用户自带 API Key。
- OpenAI-compatible：兼容 OpenAI 接口协议的模型服务。
- 本地后端：和 App 一起启动、只在本机运行的业务服务层。
- SQLite：轻量级本地数据库。
